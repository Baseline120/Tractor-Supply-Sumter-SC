# Tractor Supply Sumter SC Construction Status

Public HTML construction status page for the Tractor Supply Sumter, South Carolina project.

## Project summary
This repository holds the current one-page construction status sheet for the Sumter project, along with dated archive copies used to preserve issued versions over time.

The status page is intended to provide a clear external snapshot of:
- current phase and overall project status
- major schedule milestones
- active work and near-term priorities
- key dates and permits
- issues, watch items, and project contacts

## Repository contents
- `index.html` — current published status page
- `archive/` — dated archive snapshots of prior issued versions

## Update workflow
1. Update the project data in `index.html`.
2. Save a dated archive copy when issuing a new revision.
3. Commit and push changes so the public record stays current.

## Notes
- This repository is project-specific to Tractor Supply Sumter, SC.
- Use the archive folder to preserve issued snapshots rather than overwriting project history.
